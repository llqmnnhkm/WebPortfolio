G-Ventures 2.0

"My designs are all about user-friendliness and accessibility."

About the Project

G-Ventures 2.0 is an educational game application designed to promote environmental awareness among young learners. The game combines entertainment with educational content to create an engaging learning experience about recycling, and sustainable practices.

My Role

As the lead developer on this project, I was responsible for:

  • Game design and mechanics implementation
  • UI/UX design focused on user-friendliness for children
  • Performance optimization for various devices

Technologies Used

Unity C# Android SDK

Key Achievements

  • Successfully launched the game with positive user feedback
  • Implemented engaging game mechanics that effectively teach environmental concepts
  • Created an intuitive interface that allows children to navigate easily
  • Optimized the application for smooth performance on various devices
