e-Musafirgo

"My designs are all about user-friendliness and accessibility."

About the Project

e-Musafirgo is a comprehensive travel companion application designed specifically for Muslim travelers. The app provides essential Islamic guidance and resources for travelers, including travel dua's ,solat tutorial when musafir, halal food options, and important Islamic rulings for travel.

Project Vision

e-Musafirgo aims to make traveling easier for Muslims by providing all necessary religious resources in one convenient application. Our goal was to create a user-friendly interface that allows travelers to quickly access important information without compromising on beautiful design.

Key Features

  • Islamic Conditions: Important rulings for Muslim travelers
  • Travel Duas: Essential prayers for journey
  • Solat Tutorial: How to perform prayers while traveling
  • Halal Food Locator: Find nearby halal restaurants and food options
  • Appropriate Clothing: Guidelines for appropriate Islamic attire

My Role

As the mobile app developer for e-Musafirgo, I was responsible for the entire development process, including:

  • Designing and building the user interface (UI/UX)
  • Developing and implementing all front-end and back-end functionalities
  • Integrating location services
  • Setting up features like halal food locator, travel duas, and solat guide
  • Ensuring the app runs smoothly and provides a user-friendly experience

Technologies Used

Flutter Dart Google Maps API
